# Phishing Simulator

Güvenlik farkındalık eğitimi ve eğitim amaçları için kapsamlı bir kimlik avı simülasyon platformu.

## Genel Bakış

Phishing Simulator, kuruluşların ve bireylerin kimlik avı saldırılarını pratik, uygulamalı simülasyon yoluyla anlamalarına yardımcı olmak için tasarlanmış bir araçtır. Bu proje, gerçekçi kimlik avı senaryoları oluşturmak için HTML şablonlarını Python arka uç mantığı ile birleştirir.

## Özellikler

- 🎯 Gerçekçi kimlik avı e-posta simülasyonları
- 📊 Kullanıcı etkileşimi izleme ve analitikleri
- 🔒 Güvenli test ortamı
- 📈 Eğitim raporlaması
- 🎓 Özelleştirilebilir eğitim materyalleri ve farkındalık içeriği
- 🔗 Kişiselleştirilebilir kimlik avı senaryoları

## Teknoloji Yığını

- **Ön Uç**: HTML (87.8%)
- **Arka Uç**: Python (12.2%)

```
phishing-simulator/

```

## Kurulum

### Ön Koşullar

- Python 3.7+
- JavaScript etkinleştirilmiş web tarayıcı
- Git

### Kurulum Adımları

1. Depoyu klonlayın:
```bash
git clone https://github.com/xGhostxs/phishing-simulator.git
cd phishing-simulator
```

2. Python bağımlılıklarını yükleyin:
```bash
pip install -r requirements.txt  # varsa
```

3. Uygulamayı çalıştırın:
```bash
python app.py  # veya ana Python dosyanız
```

## Kullanım

1. Uygulamayı başlatın
2. Web arayüzüne tarayıcınız aracılığıyla erişin
3. Bir kimlik avı senaryosu oluşturun veya seçin
4. Simülasyonu test kullanıcılarına dağıtın
5. Kontrol paneli aracılığıyla sonuçları izleyin ve analiz edin

## Eğitim Amacı

Bu simülatör şu amaçlar için kullanılır:
- **Güvenlik Farkındalık Eğitimi**: Çalışanların kimlik avı girişimlerini tanımasına yardımcı olun
- **Eğitim Ortamları**: Sınıflarda siber güvenlik kavramları öğretin
- **Güvenlik Araştırması**: Kimlik avı desenleri ve kullanıcı davranışlarını çalışın
- **Yetkili Test**: Uygun yetkilendirme ile kontrollü güvenlik değerlendirmeleri yapın
